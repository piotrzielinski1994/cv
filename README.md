# CV generator

## Install dependencies

```bash
nvm use
npm install
```

## Replace example data with the final one

```bash
# Example commands
cp ./data/data.example.ts   ./data/data.ts
cp ./data/photo.example.jpg ./data/photo.jpg  # Supports only JPEG files
```

## Development

```bash
npm start
```

## Generate PDF

```bash
npm run pdf
```
